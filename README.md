# 🏠 Real Estate Listing Platform

A full-featured real estate listing web application built with **React**, **TypeScript**, and **Vite**, featuring **user authentication**, **map-based location selection**, and full **CRUD operations** for housing advertisements.

## ✨ Features

- 🔐 **User Authentication** — Register and log in with secure token-based auth using `json-server-auth`.
- 🗺️ **Map-Based Location Picker** — Select property coordinates interactively using **Leaflet** and **React-Leaflet**.
- 📝 **Post Ad Page** — Users can post new listings with:
  - Address
  - Phone Number
  - Description
  - Location coordinates
- 📄 **Ad Detail View** — View complete ad info with:
  - Map
  - Phone number
  - Full address
  - Edit and Delete buttons
- 📚 **Home Page with Pagination** — Browse all listings (address only) displayed as cards with pagination.
- 🌓 **Dark & Light Mode** — Easily switch between dark and light themes.
- 🧠 **Modern Architecture** — Built with:
  - TypeScript
  - Context API for global state
  - Custom React Hooks for clean separation of logic

## 🛠 Tech Stack

| Category   | Technologies                                    |
| ---------- | ----------------------------------------------- |
| Frontend   | React, TypeScript, Vite, React Router           |
| Map        | Leaflet, React-Leaflet                          |
| Backend    | json-server, json-server-auth                   |
| Styling    | Tailwind CSS or Styled-components (your choice) |
